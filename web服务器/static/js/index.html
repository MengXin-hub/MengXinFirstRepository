欢迎来到巴塞罗那足球俱乐部！

巴塞罗那足球俱乐部,简称巴萨，是一家位于西班牙巴塞罗那的足球俱乐部，于1899年由瑞士企业家胡安・甘珀所创立，世界球坛顶级俱乐部之一。 俱乐部主场诺坎普体育场可容纳超过十万名观众，是全欧洲最大及世界第二大的足球场。 该队百年的历史上，获得许多冠军奖杯，是西班牙赢得最多冠军奖杯的俱乐部。在1929年夺得第一届西班牙足球甲级联赛以及之后的26次联赛冠军，30次西班牙国王杯冠军，5次欧洲冠军联赛冠军及3次国际足联俱乐部世界杯冠军。 在2009年，巴塞罗那足球队连获西班牙足球甲级联赛、西班牙国王杯、欧洲冠军联赛三项冠军，成为西班牙足球史及队史上第一个三冠王球队。 接着参加下半年的三项专属于冠军之间的赛事，再度荣获西班牙超级杯、欧洲超级杯以及国际足联俱乐部世界杯三大锦标，完成世界足坛绝无仅有的唯二次“六冠王”伟业。 2011年，巴塞罗那足球队除获西班牙足球甲级联赛三连冠外，再度荣获欧洲冠军联赛冠军，5年内第3次捧起了欧冠奖杯，之后2011年下半年，又接连夺得西班牙超级杯、欧洲超级杯、以及国际足联俱乐部世界杯三大锦标，成为2011年的五冠王， 近3年内夺得了总计13座奖杯。 2015年，巴塞罗那在西甲联赛第37轮以1：0作客击败上届冠军马德里竞技提前一轮获得14-15赛季西甲联赛冠军。于西班牙国王杯决赛中，以3：1击败毕尔巴竞技夺得冠军。在欧冠决赛以3：1击败意甲尤文图斯取得俱乐部第五座欧洲冠军奖杯。 欧洲超级杯靠着佩德罗于加时赛以5：4绝杀塞维利亚夺冠；世俱杯赛事首先以苏亚雷斯上演帽子戏法3：0击败中国广州恒大，决赛同以3：0击败阿根廷河床获得2015的第五冠并成为史上第一支两次夺得三冠王及三次夺得五冠王的球队， 是二十一世纪以来最成功的俱乐部之一。 2015-16赛季，巴塞罗那缔造连续39场不败，打破88-89赛季由皇马所创下的34场不败纪录，在当季也蝉联西甲冠军及国王杯冠军(加时赛2:0击败塞维利亚)。 2019-20赛季的欧冠，巴塞罗那以2:8的成绩在欧冠四分之一决赛惨败德甲豪门拜仁慕尼黑，不仅刷新了巴塞罗那在欧冠赛场上的最多失球记录，还成为欧冠淘汰赛首支单场被轰入8球的足球俱乐部。 另外，巴塞罗那亦有自己的篮球队、五人足球队、女子足球队、手球队、冰上曲棍球队、榄球队等其他运动球队。
